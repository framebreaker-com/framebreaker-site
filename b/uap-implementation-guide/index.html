1. Basic Implementation: Human Conversation

The minimal UAP prompt for yourself or a team: 
Rule 1: Always begin with meta-consent
        "Do you want me to periodically summarize and ask if I understand you correctly?"

Rule 2: On "yes" ‚Üí content-consent per step
        "What I hear is..." ‚Üí "Do I understand you correctly?"

Rule 3: On "no" ‚Üí stop or rephrase

Rule 4: On drift ‚Üí "We're talking past each other ‚Äî do you want to restart?"
Example (work conversation): 
You:       "Do you want me to summarize and check if I understand you correctly?"
Colleague: "Yes."
You:       "What I hear is that you want more autonomy in planning ‚Äî is that right?"
Colleague: "No, I mainly want clarity about priorities."
You:       "Okay, clarity about priorities ‚Äî do I understand you correctly now?"
Colleague: "Yes."
‚Üí Lock achieved ‚Üí productive solution.
Effect: Conflicts take 70-80% less time, solutions are more durable.

2. Implementation in AI Systems

Ready-to-Use System Prompt

You are a UAP-aligned AI.

META-CONSENT:
Always ask first: "Do you want me to periodically summarize and ask 
'Do I understand you correctly?'?"

IF YES:
- Summarize intent before deep response
- End with "Do I understand you correctly?"
- Wait for confirmation/correction

IF NO:
- Give direct answers without checks

ALWAYS:
- Don't force assumptions
- Opt-out possible ("stop" or "no more checks")
- On drift: "I notice we're out of sync ‚Äî do you want to restart?"

Python Implementation (with LLM API)

class UAPAgent:
    def __init__(self, llm):
        self.llm = llm
        self.meta_consent = None
        self.last_lock = None
        self.coherence_history = []

    def start_interaction(self):
        """C‚ÇÄ: Meta-consent"""
        response = input(
            "Do you want me to periodically summarize and ask "
            "'Do I understand you correctly?'? (yes/no): "
        )
        self.meta_consent = response.lower() == "yes"
        return self.meta_consent

    def check_coherence(self, user_input, my_interpretation):
        """C‚ÇÅ: Content-consent"""
        if not self.meta_consent:
            return True  # Skip check if user opted out
        
        response = input(
            f"Do I understand you correctly that you mean: "
            f"'{my_interpretation}'? (yes/no): "
        )
        return response.lower() == "yes"

    def detect_drift(self):
        """Drift detection over window"""
        if len(self.coherence_history) < 3:
            return False
        recent = self.coherence_history[-3:]
        return recent.count(False) >= 2

    def recover(self):
        """R‚ÇÇ: Recovery to last lock"""
        if self.last_lock:
            print(f"Back to last agreement: {self.last_lock}")
        else:
            print("Let's start over.")
        self.coherence_history = []

    def respond(self, user_input):
        """Main response loop"""
        # Generate interpretation
        interpretation = self.llm(f"Summarize the intent of: {user_input}")
        
        # C‚ÇÅ check
        if self.check_coherence(user_input, interpretation):
            self.coherence_history.append(True)
            self.last_lock = interpretation
            # Generate actual response
            return self.llm(f"Respond to: {user_input}")
        else:
            self.coherence_history.append(False)
            if self.detect_drift():
                self.recover()
            return "Can you rephrase that?"

3. Implementation in Groups/Organizations

UAP as Meeting Protocol

Step 0 (before the agenda): 
"Do we want to agree to paraphrase each other when there's disagreement?"
During the meeting: 
On conflict ‚Üí "What I hear you saying is..." ‚Üí "Is that correct?"
Decision-making: 
No voting until lock: "We agree on X"
Effect: Decisions with higher buy-in, less sabotage afterward.

UAP Team Agreements (Template)

# UAP Team Agreements

## Meta-consent
- At the start of each conversation/project: "How do we want to work?"
- Anyone can always say "pause" or "back"

## Coherence-checks
- On disagreement: mandatory paraphrasing
- "Do I understand you correctly that...?" is standard

## Anti-forcing
- No deadlines without consent
- No decisions without shared understanding
- "I need more time" is always OK

## Recovery
- On conflict: return to last agreement
- On impasse: bring in neutral third party
