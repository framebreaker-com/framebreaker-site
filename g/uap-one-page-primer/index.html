What Problem Does This Solve?

Unstable interaction caused by:

  • Assumptions
  • Pressure
  • Escalation
  • Misalignment

What Is UAP?

A set of rules for how you interact with each other.

Not what is true.

The Main Rule

You may never reduce someone's options without their consent.

What Is The Endpoint?

Not consensus. Not truth. Not winning.

But:

Nothing needs to be forced anymore.

The Two Questions

1. "Do you want this conversation?" (Consent to begin)

2. "Do I understand you correctly?" (Check per step)

When Does It Work?

  • When both parties can stop
  • When "no" is a real option
  • When nobody pushes

When Does It Fail?

  • When forcing is permitted
  • When consent is simulated
  • When coherence isn't checked

Is This Soft?

No. It only forbids coercion, not sharpness.

Is This Slow?

Faster than escalation + repair.
